# DineDrive

## Run locally

1. Open the repository in your editor.
2. Serve `DineDrive (2)/DineDrive` with any static server (for example, VS Code Live Server).
3. Open `index.html`.

## Hosting (GitHub Pages)

This repository includes a GitHub Actions workflow at:

`.github/workflows/deploy-pages.yml`

When changes are pushed to `main` or `master`, the workflow deploys the static site from:

`DineDrive (2)/DineDrive`

If GitHub Pages is not already enabled, the deployment workflow attempts to enable it automatically.
